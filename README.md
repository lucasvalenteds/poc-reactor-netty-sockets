# POC: Reactor Netty WebSockets

It demonstrates Non-blocking web server based on WebSockets running on Netty with Reactor. The examples include how to handle headers and multiple flows of communication (client-server, server-client and duplex).

| :--- | :--- |
| Run tests | `./gradlew test` |
